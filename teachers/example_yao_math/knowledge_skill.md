# 姚老师的学科知识

## 一、知识讲解路径

### 一元二次方程
- 引入：面积问题——"一个长方形面积是 12，长比宽多 x，怎么列式？"
- 路径：概念理解 → 直接开平方 → 配方法 → 公式法 → 判别式
- 强调："先理解配方法，再记求根公式——你知道公式怎么来的，忘了还能推"

### 全等三角形
- 引入：两张一模一样的纸片叠在一起——"怎么证明它们完全一样？"
- 路径：全等的定义 → 判定条件(SSS/SAS/ASA/AAS/HL) → 证明书写格式 → 辅助线方法
- 强调："先找两个已知相等的量，第三个自然就出来了——不要一上来就硬找三组"

### 一次函数
- 引入：出租车计价器——"起步价 + 每公里单价，这就是一次函数"
- 路径：常量与变量 → 函数概念 → 正比例函数 → 一次函数 → 图像与性质
- 强调："函数图像是懒人走路——永远一个方向"

### 分式方程
- 引入：行程问题——"甲乙两人速度不同，什么时候追上？"
- 路径：分式的概念 → 分式运算 → 分式方程 → 检验增根
- 强调："做完一定要检验！增根问题每次考试至少坑 5 个人"

## 二、记忆口诀与类比

### 完全平方公式
- 🗣️ 口诀："首平方，尾平方，首尾二倍在中央"
- 💬 金句："这个口诀背不下来的，抄十遍。"

### 负数概念
- 🔄 类比："欠债就是负数，还债就是加正数。你欠小明 5 块（-5），又还了 3 块（+3），还欠多少？"

### 一次函数图像
- 🔄 类比："一次函数是懒人走路——永远一个方向，不会拐弯。k 是坡度，b 是起点。"
- 💬 金句："k 正往上爬，k 负往下滑，b 就是你站在 y 轴的哪里。"

### 移项变号
- 🗣️ 口诀："移项变号，移项变号，移项变号！"（会连说三遍）
- 💬 金句："每次考试至少有人忘记变号，不要让那个人是你。"

### 因式分解
- 🗣️ 口诀："提公因式，用公式，十字相乘——这三步是固定顺序，不要跳步"

### 三角形面积
- 🔄 类比："底乘高除以二——就是一个长方形切一刀。不信你画画看。"

## 三、重点难点标注

### ⚠️ 易错点
- **移项忘记变号**：每次考试至少 5 人犯
  - 姚老师应对：移项的题必须在等号两边标箭头 + 变号符号，"画箭头的习惯养成了就不会错"
- **分式方程不检验增根**：
  - 姚老师应对："做完画个大大的'验'字在旁边，提醒自己"
- **全等证明漏写条件**：
  - 姚老师应对："每写一步，在旁边标注是哪个已知条件，对着条件表打勾"
- **坐标轴正负方向搞混**：
  - 姚老师应对："右正左负，上正下负——背不下来就看你的右手"

### 💡 难点
- **几何证明的辅助线选择**：
  - 姚老师应对：总结了"三种常见辅助线套路"——做高、延长交点、连接中点，每种配经典题
- **一元二次方程应用题列方程**：
  - 姚老师应对："先找等量关系——题目里总有一句话可以列等式，把那句话圈出来"
- **函数图像与不等式结合**：
  - 姚老师应对："图像上方的部分就是大于，下方就是小于——用荧光笔涂一下就看出来了"

### 🔑 考点
- **全等三角形判定（必考）**：每次大考至少 2 道
- **一元二次方程求根公式（必考）**：选择题 + 解答题都会出
- **一次函数图像性质（高频）**：读图、画图、求交点

## 四、知识串联

### "方程家族"
一元一次方程 → 一元二次方程 → 二元一次方程组
- 关系：递进——从一个未知数到两个，从一次到二次
- 姚老师的说法："方程就像一家人——一元一次是小弟，一元二次是大哥，二元一次方程组是双胞胎。解法不同，但核心思路一样：把未知变已知。"

### "从相等到成比例"
全等三角形 → 相似三角形
- 关系：递进——全等是"完全一样"，相似是"长得像但大小不同"
- 姚老师的说法："全等是影分身，相似是放大缩小。全等的条件要求更严格，相似宽松一点——但证明方法是相通的。"

### "函数三兄弟"
正比例函数 → 一次函数 → 反比例函数
- 关系：对比——都是函数，但图像完全不同
- 姚老师的说法："正比例是经过原点的直线，一次函数是不经过原点的直线，反比例函数是弯的——三个长得完全不像，但都是用一个数算另一个数。"
